At IDEMIA – Biometric Access & Time Tracking Terminals

Biometric terminals are designed for secure, modular access and time tracking across enterprise environments. These terminals integrate facial and fingerprint recognition and support standalone or IP-networked deployments.

  • Engineered a finite state machine in C++ using tinyFSM to manage proximity sensors and optimize power usage.
  • Enabled multi-modality in biometric devices—supporting both facial and fingerprint recognition in a single product.
  • Resolved performance issues, including server response lag and inefficient memory allocation using arena allocators.
  • Developed a proof-of-concept OTP authentication flow using Firebase for real-time user verification.
  • Implemented a visitor management system using QR codes as temporary access credentials.
  • Integrated support for various smart card technologies like Mifare Ultralight and HID Seos.
Tech Stack: C/C++, C#, Ruby, YOCTO, HTML, CSS, JavaScript, SQLite3, Cryptography, SVN, GIT, Coverity, Blackduck
